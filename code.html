HTML приклад

Тег <h1> використовується для заголовків першого рівня.

                
<!DOCTYPE html>
<html lang="uk">
    <head>
        <title>Моя сторінка</title>
    </head>
    <body>
        <h1>Вітаю!</h1>
    </body>
</html>

Python приклад

                
def calculate_sum(a, b):
    """
    Функція обчислює суму двох чисел
    """
    result = a + b
    return result

# Виклик функції
total = calculate_sum(5, 3)
print(f"Сума: {total}")
